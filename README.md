# ChatEverything

ChatEverything 是一个专业的 AI 英语对话助手，支持自定义大语言模型，让英语学习和练习更有趣、更高效。

## 主要特点

### 🤖 自定义大模型
- 支持多种主流大语言模型
- 可根据需求自由切换和配置
- 灵活的模型参数调整

### 🎯 场景化对话
- 提供丰富的英语对话场景
- 帮助提升口语和写作能力
- 真实情境模拟练习

## 功能预览

ChatEverything 提供多样化的功能：

- 智能对话界面
- 丰富的场景选择
- 灵活的模型配置
- 个性化设置
